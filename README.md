# Tumor Segmentation Using Deep Learning

Desktop-based deep learning project for breast tumor segmentation, classification, and explainability.

## Scope
- Breast image preprocessing
- Tumor / lesion segmentation
- Benign vs malignant classification
- Explainability visualization
- Desktop interface

## Project Structure
- `docs/` documentation
- `src/` source code
- `tests/` test files
- `configs/` configuration files
- `scripts/` helper scripts
- `data/` dataset folders
- `notebooks/` experiments

## Current dataset can be found on Kaggle
https://www.kaggle.com/datasets/abdelrahmanelmugh/cbis-ddsm-512-full1-wm
